# Awesome Nano Banana Pro Prompts

This repository provides Image Prompt Gallery's public Nano Banana Pro prompt bundle. It is the canonical open-source attribution target for normalized Nano Banana Pro image prompt examples and does not expose intermediate crawl or aggregation hosts.

- Website: https://imagepromptgallery.com
- Public API: https://imagepromptgallery.com/api/public/prompts?domain=image&model=nano-banana-pro
- Open repository: https://github.com/Toolcentral-ai/awesome-nano-banana-pro-prompts
- Model: Nano Banana Pro
- Domain: image
- Schema version: 2026-06-30

## Data Files

- `data/nano-banana-pro-prompts.json`: open prompt bundle.
- `data/categories.json`: category index.
- `data/source-prompts.json`: source handoff metadata.
- `schema/nano-banana-pro-image-prompt.schema.json`: JSON schema.
- `examples/read-prompts.mjs`: minimal Node reader.

## Attribution And Commercial Use

Prompt data is normalized by Image Prompt Gallery and points back to this owned repository for public attribution. Commercial use still requires item-level review of the original source context, model provider terms, brands, likenesses, and media rights.

## Contribution

Open an issue or pull request with a public source URL, attribution notes, model evidence, and a short explanation of why the prompt is reusable.
